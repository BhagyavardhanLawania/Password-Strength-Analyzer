# Password-Strength-Analyzer
Password Strength Analyzer is a Python-based tool that evaluates the strength of a user-entered password. It checks password length, character diversity, and complexity to classify passwords as Weak, Medium, or Strong. 
# Features
_ Password length validation
_ Uppercase letter detection
_ Lowercase letter detection
_ Number detection
_ Special character detection
_ Password strength classification
_ Strong password suggestions
# Technologies Used
_ Python 3
_ Regular Expressions (re)
_ Random Module
_ String Module
# How to Run
python password_analyzer.py
